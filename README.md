# シンプルタイマー

シンプルで使いやすいウェブベースのタイマーアプリケーションです。

## 機能

- 分と秒の指定が可能
- よく使う時間（3分、5分、10分、25分、40分）を素早く設定できるプリセットボタン
- タイマー終了時にメロディが再生
- シンプルで直感的なUI

## 使い方

1. 分と秒を入力するか、プリセットボタンをクリックして時間を設定
2. 「開始」ボタンをクリックしてタイマーをスタート
3. 「停止」ボタンで一時停止、「リセット」ボタンで初期状態に戻すことができます

## 技術スタック

- HTML5
- CSS3
- JavaScript (ES6+)
- Web Audio API
